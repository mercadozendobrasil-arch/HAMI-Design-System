# Assets

本目录存放 HAMI 的固定设计资产。

```text
assets/
├── logo/          HAMI Logo，优先 SVG/透明 PNG
├── icons/         统一线性 SVG 图标
├── backgrounds/   固定银灰背景
└── products/      按产品系列整理的透明 PNG
```

## Product Asset Requirements

- 透明背景 PNG
- 建议原图边长不低于 1600 px
- 不带文字、水印和非产品装饰
- 每款产品建议提供正面、背面、45°、支架状态和细节图
- 文件名使用英文、数字和连字符

示例：

```text
products/universal-360/light-blue/front.png
products/universal-360/light-blue/back.png
products/universal-360/light-blue/stand.png
```
