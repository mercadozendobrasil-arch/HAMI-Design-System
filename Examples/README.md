# Examples

## Purpose

保存通过审核的引用方式和命令解析示例，不保存虚构商品素材。

## Usage

```text
Generate QT Redmi Pad 2 Shopee Cover
→ series: QT
→ channel: Shopee
→ template: Template001-Cover
→ product: 必须由 Product Library 唯一匹配
→ QA: Cover + ProductLock + Layout + Final
```

如果真实产品记录不存在，示例结果必须是 `BLOCKED_MISSING_PRODUCT_RECORD`。任何展示图只有在正式产品资产、metadata、image manifest 和 Final QA 全部存在后才能加入本目录。
