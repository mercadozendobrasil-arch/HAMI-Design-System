# Typography

## Font Family

默认字体：`Montserrat`。

仅在系统无法使用 Montserrat 时，按顺序回退：`Arial`, `sans-serif`。

## Type Scale for 800×800 Images

| Role | Size | Weight | Line height |
|---|---:|---:|---:|
| H1 | 52–64 px | 700 | 1.05 |
| H2 | 38–46 px | 700 | 1.10 |
| Subtitle | 24–30 px | 600 | 1.20 |
| Feature label | 20–24 px | 600 | 1.15 |
| Caption | 16–18 px | 500 | 1.20 |

## Rules

- 每张图只使用一个字体家族。
- 标题最多两行，副标题最多一行。
- 不得使用衬线字体、描边字、3D 字或夸张投影。
- 文案过长时删减内容，不得把字号缩到手机端不可读。
- 巴西葡语必须保留正确重音符号。
