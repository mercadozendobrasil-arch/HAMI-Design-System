# Prompt Library

## Purpose
提供薄编排入口，把用户意图路由到固定模板；prompt 不包含视觉规则。

## Usage
先执行 `PromptContract.md`，再选择 Prompt001–009。所有视觉值始终从对应 Design System 模块读取。

## Dependencies
`../CODEX.md`, `../Templates/`, `../QA/`, Product Library target record.
