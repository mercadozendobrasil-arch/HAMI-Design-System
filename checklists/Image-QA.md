# HAMI Image QA Checklist

每张图片导出前必须逐项检查。

## Product
- [ ] 产品未重绘、拉伸、压扁或补画
- [ ] 摄像头孔位、按键、接口、扣子和支架结构正确
- [ ] 产品颜色与真实商品一致
- [ ] 产品边缘清晰，没有抠图毛边或白边
- [ ] 关键结构未被裁切或遮挡

## Brand
- [ ] 使用 HAMI 规定的银灰/白背景
- [ ] Logo 位置和尺寸符合模板
- [ ] 仅使用 Montserrat 或规定回退字体
- [ ] 图标来自同一线性图标体系
- [ ] 阴影自然且与其他图片一致

## Copy
- [ ] 巴西葡语拼写和重音正确
- [ ] 每个文案块不超过两行
- [ ] 没有无价值的重复文案
- [ ] 手机端缩小后仍清晰可读
- [ ] 型号、尺寸和功能声明均已确认

## Commerce
- [ ] 没有展示未包含的配件
- [ ] 没有第三方品牌 Logo
- [ ] 没有误导性认证、参数或夸张承诺
- [ ] 主图能在 1 秒内看懂产品和核心卖点
- [ ] 输出为 800×800 RGB PNG

任一强制项不通过时，不得导出最终文件。
