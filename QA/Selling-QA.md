# Selling QA

## Purpose
Reject unverified, crowded or stylistically inconsistent selling-point pages.

## Usage
Apply this checklist to every Template002 output before Final QA.

- [ ] Template002 loaded.
- [ ] One unchanged product and no unrelated scene.
- [ ] At most four verified selling points.
- [ ] Every icon is an approved outline icon.
- [ ] Labels are concise Brazilian Portuguese and do not obscure key structures.

## Dependencies / Related Modules / Version
Template002 and Final-QA. Related to Selling Point and Icon components. Version `2.0.0`.
