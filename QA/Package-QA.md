# Package QA

## Purpose
Reject package pages that imply unverified items or quantities.

## Usage
Apply this checklist to every Template009 output before Final QA.

- [ ] Template009 loaded.
- [ ] Every visible item and quantity exists in package metadata.
- [ ] No device, stylus, film or accessory is implied unless included.
- [ ] Real asset provenance exists for every displayed item.
- [ ] Quantity labels are single-line Brazilian Portuguese.

## Dependencies / Related Modules / Version
Template009, Product Library schema and Final-QA. Related to Package prompt. Version `2.0.0`.
