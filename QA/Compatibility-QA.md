# Compatibility QA

## Purpose
Reject inaccurate or unreadable compatibility pages.

## Usage
Apply this checklist to every Template008 output before Final QA.

- [ ] Template008 loaded.
- [ ] Every model, year and size matches product metadata.
- [ ] No inferred model or third-party Logo appears.
- [ ] Each Compatibility Card contains at most five models.
- [ ] Minimum type size and line limits pass.

## Dependencies / Related Modules / Version
Template008, Product Library schema and Final-QA. Related to Compatibility Card. Version `2.0.0`.
