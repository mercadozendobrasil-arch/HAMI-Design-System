# Layout QA

- [ ] Every resolved bound is on the grid or declared by a token.
- [ ] Essential content remains inside the safe area.
- [ ] Logo, text, product and footer occupy named template zones.
- [ ] Copy does not overlap the product.
- [ ] Layer order and spacing use tokens.
- [ ] Product coverage is within global limits.

Dependencies: Layout Rules and layout tokens. Version `2.0.0`.
