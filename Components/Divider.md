# Divider Component

## Purpose
Separate information groups without decorative noise.

## Usage / Inputs
`orientation=horizontal|vertical`, optional `lengthToken`.

## Slots
One hairline rule.

## Tokens
`border.hairline`, `divider.gap`.

## QA
Reject arbitrary thickness, accent colors or intersections with product-safe bounds.

## Dependencies / Related / Version
`ComponentContract.md`; used by Compatibility and Package layouts. Version `2.0.0`.
