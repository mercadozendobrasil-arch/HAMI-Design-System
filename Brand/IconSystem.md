# Icon System

## Style

- 单一线性图标风格
- 圆角端点
- 2–3 px 视觉线宽（800×800 画布）
- 图标颜色默认 `#111111` 或 `#1769C2`
- 不混用填充、拟物、3D 和卡通图标

## Standard Size

- 封面图标：52–60 px
- 辅图功能图标：64–88 px
- 图标与文字间距：12–16 px

## Core Library

- Rotação 360°
- Proteção contra impactos
- Ajuste universal
- Encaixe seguro
- Suporte ajustável
- Auto Sleep/Wake
- Porta-caneta
- Alta transparência
- Dureza 9H
- Recortes precisos

图标必须来自同一套 SVG 资源。若资源不存在，应先建立图标，再用于全部模板。
