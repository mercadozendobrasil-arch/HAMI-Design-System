# Voice and Copy

## Purpose

Define concise Brazilian Portuguese copy that communicates verified product value without hype.

## Voice

- Professional, direct and reassuring.
- Feature first, user benefit second.
- Prefer concrete nouns and active verbs.
- Preserve correct Portuguese accents.

## Usage

- One primary message per image.
- Never invent compatibility, certification, material, dimensions or included accessories.
- Avoid `100%`, `indestrutível`, `o melhor`, urgency language and unsupported superlatives.
- Product names, model years and screen sizes must come from product metadata.
