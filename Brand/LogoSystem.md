# Logo System

## Purpose

Keep the HAMI signature consistent across every official-store image.

## Canonical asset

Use the approved SVG or transparent PNG under `Assets/logo/`. Never typeset a substitute, redraw the mark or use a generated logo.

## Usage

- Default position: top-left at `layout.safe.margin` coordinates.
- Cover width: `logo.cover.width`.
- Clear space: at least `logo.clearSpace` on all sides.
- Use the dark mark on silver/white backgrounds; use the inverse mark only on an approved dark surface.
- Do not rotate, stretch, recolor, outline or add effects.
