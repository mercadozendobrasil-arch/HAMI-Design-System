# Color System

## Core Palette

| Token | HEX | Usage |
|---|---|---|
| `hami-black` | `#111111` | 主标题、Logo、核心信息 |
| `hami-gray-700` | `#4F545C` | 副标题、说明文字 |
| `hami-gray-400` | `#AEB4BC` | 分隔线、次要图形 |
| `hami-silver-100` | `#F7F8FA` | 背景顶部 |
| `hami-silver-200` | `#E8EBEF` | 背景底部 |
| `hami-white` | `#FFFFFF` | 卡片与高光 |
| `hami-blue` | `#1769C2` | 功能强调、少量图标 |

## Default Background

封面默认使用垂直银灰渐变：

```text
Top:    #F7F8FA
Bottom: #E8EBEF
```

允许非常轻微的工作室地面反射，但不得加入复杂纹理。

## Usage Rules

- 单张图片最多一个强调色。
- 强调色面积原则上不超过画面的 10%。
- 产品本身颜色不计入品牌强调色限制。
- 不得为了活泼感随机加入紫色、橙色、红色等。
- 黑色文字优先，灰色仅用于次级内容。
