# HAMI Outline Icon Library

十个核心图标统一使用 64×64 viewBox、`currentColor`、2.5 px 视觉线宽、圆角端点和圆角连接。调用时不得改写 path，也不得混入其他图标风格。

| Asset | Meaning |
| --- | --- |
| `rotation-360.svg` | Rotação 360° |
| `impact-protection.svg` | Proteção contra impactos |
| `universal-fit.svg` | Ajuste universal |
| `secure-fit.svg` | Encaixe seguro |
| `adjustable-stand.svg` | Suporte ajustável |
| `sleep-wake.svg` | Auto Sleep/Wake |
| `stylus-holder.svg` | Porta-caneta |
| `high-transparency.svg` | Alta transparência |
| `hardness-9h.svg` | Dureza 9H |
| `precision-cutouts.svg` | Recortes precisos |
