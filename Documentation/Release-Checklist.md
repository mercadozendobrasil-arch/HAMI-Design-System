# Release Checklist

## Purpose

在发布设计系统版本前提供一致的验收门禁。

## Usage

- [ ] 根 manifest、README 和模块版本一致。
- [ ] 所有 JSON 可解析，schema 文件有效。
- [ ] Template001–009 均含全部必需章节。
- [ ] 模板没有复制组件或硬编码布局值。
- [ ] Prompt 只引用设计系统契约。
- [ ] QA 单元测试通过。
- [ ] SVG 可解析且符合 outline 契约。
- [ ] Markdown 相对链接均可解析。
- [ ] Product Library 兼容版本已声明。
- [ ] 工作树干净且远端 `main` 与本地 HEAD 一致。
