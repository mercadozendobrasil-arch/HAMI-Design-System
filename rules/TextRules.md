# Text Rules

## Default Language

面向巴西电商的图片文案默认使用巴西葡萄牙语。

## Maximum Copy

- 每个独立文案块最多两行。
- 非必要情况下优先一行。
- 封面最多：一个标题、一个副标题、四个短卖点。
- 辅图原则上只表达一个核心卖点。

## Writing Style

- 使用短词组，不写段落。
- 先写功能，再写结果。
- 避免绝对化声明，例如 `100%`, `indestrutível`, `o melhor`。
- 不写无法证明的参数。
- 型号与尺寸必须准确。

## Preferred Examples

- `Rotação 360°`
- `Ajuste Universal`
- `Proteção Completa`
- `Encaixe Seguro`
- `Alta Transparência`

## Avoid

- 重复同一卖点
- 超过两行的说明
- 为填满空间增加无价值文案
- 过小字体
